## FULLSTACK WEBSITE FOR A FICTIONAL CAFE

**Frontend tools used:**

- reactjs
- react router
- redux-toolkit
- axios
- gsap
- tailwindcss

**Backend tools used:**

- nodejs
- express
- mongodb
- multer

---

#### Project launch

**Download the necessary modules to start the project:**

- npm i

**Enter the command**

- npm start

---

**YOU MUST HAVE MONGODB INSTALLED TO USE THE DATABASE!**
